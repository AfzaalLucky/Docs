Members  
=======  

| Member        | Value | Description                                                                    |
|---------------|-------|--------------------------------------------------------------------------------|
| **Automatic** | 0     | The beam angle is automatically adjusted to point to the loudest sound source. |
| **Manual**    | 1     | The beam angle is fixed and can be set by the application.                     |
